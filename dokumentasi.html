📱 Panduan Penggunaan & Modifikasi PWA

1. Cara Mengganti Ikon PWA (Muncul di Layar HP)

  1. Siapkan gambar logo Anda (harus rasio persegi 1:1, misalnya logo restoran Anda).
  2. Simpan 2 file gambar berformat PNG: icon-192x192.png dan icon-512x512.png.
  3. Letakkan kedua gambar tersebut di dalam folder assets/ (buat folder ini jika belum ada).
  4. Saat pengguna menginstall web melalui browser (Add to Home Screen), ikon ini akan otomatis dipakai.

2. Cara Cetak Otomatis (Auto-Print)

  1. Buka menu Pengaturan.
  2. Nyalakan saklar (toggle) Cetak Otomatis.
  3. Sekarang, setiap kali Anda selesai menekan "Konfirmasi Bayar", dialog printer bawaan browser akan langsung muncul secara otomatis.

💻 Panduan Folder CodeCanyon (Developer)

Untuk kemudahan distribusi di CodeCanyon, aplikasi ini saat ini digabung menjadi 1 file HTML. Anda bisa dengan mudah memecahnya berdasarkan komentar yang telah saya sediakan:

1. Ekstraksi CSS

Cari baris <!-- FILE: source/assets/css/style.css START --> di dalam index.html. Cut seluruh blok <style> tersebut dan pindahkan ke file assets/css/style.css baru. Lalu panggil menggunakan <link rel="stylesheet">.

2. Ekstraksi JavaScript

Scroll ke paling bawah file, temukan <!-- FILE: source/assets/js/app.js START -->. Cut seluruh blok <script> tersebut dan simpan sebagai assets/js/app.js.

3. Offline Libraries

Di bagian <head>, Anda akan melihat link CDN untuk Tailwind, SheetJS, dll. Jika pembeli Anda ingin 100% offline murni, mintalah mereka mendownload link tersebut ke komputer mereka dan letakkan di dalam folder assets/js/ sesuai nama yang tertera di komentar.

🚀 Cara Hosting / Instalasi

  1. Upload 3 file utama: index.html, manifest.json, dan sw.js ke layanan hosting statis (misal: GitHub Pages atau Vercel).
  2. Setelah dihosting (HTTPS wajib), buka URL aplikasi di browser Chrome/Safari di HP Anda.
  3. Pilih opsi browser "Tambahkan ke Layar Utama".
  4. Selesai. Aplikasi sudah terinstal dan bisa dipakai offline kapan saja.
